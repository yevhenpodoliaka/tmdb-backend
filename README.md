### register :pencil2::pencil2::pencil2:
--------------------------------------------------------------------
`/api/auth/register`
-----------------------------------------------------------
* method:POST
* body: 
    ``` js
        {
        name:"userName",*
        email:"user@gmail.com",*
        password:password*
        } 
* response :   
  ``` js  
    {
        name: string,
        email: string,
        token: string,
        movies: [],
 
    }
  ```
-------------------------------------

## MOVIES
-----------------------------------------------------
### add :pencil2:
---------------------------------------------------
`/api/movies`
------------------------------------------------
* method:POST
* body:
``` js 
   { movieId:string*,
     group:"watched"|"favorites"|"queued",*
    }
    
```

* response :
``` js 
     {
     movies:movie[]
      }
  
 ```
    
---------------------------------------
### delete movie :wastebasket:
-----------------------------------------
`/api/movies/:_id`
------------------------------------------
* method:DELETE
* response:
``` js
  {
  
    message: "movie deleted",
    movies:movie[]
  }
```
---------------------------------------
### update movie :pencil2:
----------------------------------------
`/api/movies/:_id`
-------------------------------------------
* method:PATCH
* body:
``` js
  { group:newGroup*} ("watched"|"favorites"|"queued",*)
 

```
* response: 
``` js
 {
    message:"movie updated",
  movies:movie[]
  }
```
------------------------------------------
